# EventFlow 🚀

EventFlow is a lightweight event registration web app built during the IIT Bhilai Hackathon.

## Problem
Event registrations are often collected from WhatsApp, email, and forms and then manually managed in spreadsheets. This leads to duplicate entries and wasted effort.

## Solution
EventFlow automates the registration workflow:
- Simple web registration page
- Automatic storage in Google Sheets
- Duplicate registration prevention
- Instant confirmation emails using Gmail

## Tech Stack
- HTML, CSS, JavaScript
- Google Apps Script (Backend)
- Google Sheets (Database)
- Gmail Automation
